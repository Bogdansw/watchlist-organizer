# Movie Playlist Tracker

O pagină web simplă unde utilizatorii pot organiza filme și seriale într-o bibliotecă personală. Poți crea una sau mai multe liste, adăuga filme sau seriale și folosi categorii pentru a le organiza mai ușor.

Filmele și serialele pot avea taguri precum:
- watching
- rewatching
- dropped
- favorite
- paused
- planned

Aplicația folosește API-ul de la The Movie Database pentru a găsi informații despre filme și seriale.

## Cum se deschide

Deschide `index.html` în browser. Nu e nevoie de instalare.
